# BSTweaker 更新日志

## v1.0.2 (2026-02-05)

### 即时热重载

- `/bstweaker reload` 命令可在游戏内即时更新纹理和模型
- 无需重新加载资源包，无需重启游戏
- 标准纹理约 20ms（对比 F3+T 的 2-3 秒）
- 配置选项 `enableFastReload` 可按需关闭
- 注意：复杂动画纹理仍需使用 F3+T

### 全自动生成

- 模型、语言文件、提示信息自动生成
- 只需准备：`weapons.json` + 纹理文件
- 两行 JSON 即可创建一把武器

### 5 把绿宝石范例武器

- 匕首、战锤、长矛、战斧、双截棍
- `weapons.json` 中包含可直接使用的模板

### 未来计划

- 更多脚本效果
- 静默热重载实现武器升级效果
- 特殊武器与独特能力
- 自定义属性（如：长矛投掷不消耗概率）
- 可变形武器（多种模式切换）
- 混合能力自定义武器
- 游戏内 GUI 创建武器（无需手写 JSON）
- 为其他模组提供注册武器的 API
- 单武器能力微调（如：缴械概率）
- GeckoLib 高级模型支持
